Sobre el proyecto

Hogwarts Castle es un proyecto web académico desarrollado con HTML5, CSS3 y JavaScript puro, sin el uso de frameworks ni librerías externas.

El proyecto consume datos en tiempo real de la Harry Potter API , mostrando información de personajes, casas y hechizos del universo creado por J.K. Rowling.

La web incluye tres secciones principales: el Gran Libro (explorador de personajes), el Sombrero Seleccionador (test interactivo de casa) y el Duelo de Magos (comparativa de personajes con grimorio de hechizos).

Proyecto liberado por la creatividad y el añoro por el mundo mágico, con fines exclusivamente educativos y de práctica en el desarrollo web. Inspiración en las facetas de la saga de Harry Potter donde se quería crear un camino por su universo con metodologías deseadas en el proyecto para el conocimiento de recorrer y extraer datos de una API, así como el diseño y maquetación de una web con HTML y CSS, y la interacción con JavaScript.
